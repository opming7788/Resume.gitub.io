關於這份申請網頁，分享一個開發時的技術心得

在開發這份申請網頁的過程中，我注意到 CSS 的設定具有優先順序。看來，優先順序是以 <div class=”mytext” style=”….”> 元素內的內聯樣式為最優先，接著是 <head> 標籤中的 CSS 樣式 <style>….</style>為第二優先，最後則是獨立於 HTML 檔案外的 CSS 檔案 <link rel="stylesheet" href="mytest.css" />為最後優先。這件事給我帶來了不小的困擾，因為在開發初期我並不知道這層優先順序的存在。在測試 <h1> 標籤中的 CSS 樣式 <h1 style=” …”> 效果時，忘記刪除該標籤中的css樣式，後來在獨立於 HTML 檔案外的 CSS 檔案作設定時卻發現對該 <h1> 標籤居然沒有影響。這讓我花了很長的時間進行除錯，而後發現 CSS 樣式設定具有優先順序。這次經驗提醒我在開發過程中要更加謹慎和周詳。
